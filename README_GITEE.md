# 太阳能数据爬虫项目

这是一个用于爬取太阳能数据的Python项目，支持GitHub和Gitee双仓库同步。

## Gitee流水线设置步骤

### 1. 启用流水线

由于Gitee流水线不会自动创建，您需要手动启用它：

1. 进入Gitee仓库主页
2. 点击顶部导航栏中的「流水线」标签
3. 点击「启用流水线」按钮

### 2. 配置环境变量（可选）

如果您需要在流水线中使用环境变量（如访问令牌）：

1. 进入仓库「管理」->「部署公钥与凭据」->「凭据管理」
2. 点击「添加令牌」，创建一个具有仓库读写权限的访问令牌
3. 令牌名称：`GITEE_TOKEN`
4. 权限：至少勾选「仓库权限」->「读权限」和「写权限」

### 3. 手动触发流水线

启用流水线后，您可以：

1. 点击「手动触发」按钮运行流水线
2. 查看流水线执行日志
3. 检查数据是否成功更新

## 项目结构

```
data/             # 存储爬取的JSON数据文件
main.py          # 主爬虫脚本
requirements.txt # Python依赖列表
.gitee-ci.yml    # Gitee流水线配置文件
.github/workflows/ # GitHub Actions配置目录
```

## 本地运行

```bash
# 安装依赖
pip install -r requirements.txt

# 运行爬虫
python main.py
```
